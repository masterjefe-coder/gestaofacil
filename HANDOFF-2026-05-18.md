# Handoff 2026-05-18

## Resumo executivo

O projeto saiu de ideia/documentacao e virou uma base funcional em `Next.js`.

Hoje o Gestao Facil ja tem:

- landing page de produto
- login protegido com `next-auth`
- dashboard protegido por sessao
- setup de workspace e empresa
- CRUD local de clientes
- CRUD local de orcamentos
- CRUD local de cobrancas
- cobranca criada a partir de orcamento aprovado
- schema inicial em Prisma

Ainda nao ha banco real conectado ao app, mas a estrutura ja foi preparada para isso.

## Stack atual

- `Next.js 16`
- `React 19`
- `TypeScript`
- `next-auth` estavel para autenticacao
- `Prisma` para schema e futura persistencia real
- persistencia local temporaria em arquivo JSON

## Como rodar

1. Instalar dependencias

```bash
npm install
```

2. Criar `.env.local` a partir de `.env.example`

3. Rodar o projeto

```bash
npm run dev
```

4. Validar

```bash
npm run typecheck
npm run lint
npm run build
```
