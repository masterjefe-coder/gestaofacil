# 30 / 60 / 90 Day Roadmap

## Days 1-30

Goal: fechar proposta de valor e criar o esqueleto funcional do produto.

- consolidar nicho inicial
- validar copy principal
- definir estrutura do banco
- criar autenticacao
- criar cadastro da empresa
- criar cadastro de clientes e servicos
- criar pipeline simples
- criar fluxo de orcamento
- criar dashboard basico

Exit criteria:

- usuario entra no sistema
- cadastra empresa
- cadastra cliente
- cria orcamento
- enxerga o funil minimo

## Days 31-60

Goal: conectar venda e cobranca.

- converter orcamento em pedido
- criar area de cobrancas
- integrar Pix ou link de pagamento
- criar status de recebimento
- criar lembretes operacionais
- medir tempo ate primeira cobranca

Exit criteria:

- usuario consegue vender
- gerar cobranca
- acompanhar pagamento
- visualizar pendencias

## Days 61-90

Goal: fechar o fluxo com fiscal e preparar beta pago.

- integrar NFS-e
- reaproveitar dados da venda na emissao
- criar centro de ajuda
- adicionar IA utilitaria
- ajustar onboarding
- fechar primeiros usuarios beta

Exit criteria:

- usuario conclui fluxo completo
- venda vira cobranca
- cobranca vira recebimento
- recebimento ou entrega vira nota
- sistema ja parece um produto cobravel
